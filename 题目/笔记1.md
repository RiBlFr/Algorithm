[洛谷P1147：连续自然数的和](https://www.luogu.com.cn/problem/P1147)

## 思路
**枚举+数学推理剪枝**

给定整数 M，找出所有 连续正整数区间 [L, R]（至少两个数），使得：
L+(L+1)+⋯+R=M
连续区间长度设为 k（至少为 2），起点为 L，则：L+(L+1)+⋯+(L+k−1)=M；求和公式：kL+2k(k−1)​=M;化简：L=[M−2k(k−1)]​​/k；因此对所有 k 从 2 开始枚举，一直到2k(k-1)>=M进行剪枝即可。

```cpp
int main() {
	int n;
	cin >> n;
	vector<vector<int>>ans;
	for (int k = 2; k * (k + 1) / 2 < n; ++k) {
		long long a = (1 - k) * k / 2 + n;
		if (a <= 0)continue;
		if (a % k == 0) {
			int left = a / k;
			int right = left + k - 1;
			ans.push_back({ left,right });
		}
	}
	sort(ans.begin(), ans.end(), cmp);
	for (auto v : ans) {
		cout << v[0] << ' ' << v[1] << endl;
	}
	return 0;
}
```
